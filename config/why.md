# config/ (O Painel de Controle) 🎛️
Esta pasta serve para centralizar configurações que mudam entre um ambiente e outro ou constantes globais.

O que deveria ter:
Definições de cores do tema (se quisermos mudar o "verde Ploc" em um só lugar).
Configurações de limites de API (ex: "tempo máximo de espera do chat").
Veredito: Pode ser redundante agora, já que estamos usando muito o Shared Layer dentro de /apps/frontend. Mas é um bom lugar para guardar arquivos JSON de configuração que o Backend e o Frontend compartilham.
