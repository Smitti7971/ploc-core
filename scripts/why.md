scripts/ (A Oficina de Automação) 🛠️
Esta é a pasta para "ferramentas de bastidores".

O que deveria ter: Scripts de automação que não fazem parte do app, mas ajudam você. Por exemplo:
Um script para limpar o banco de dados e recomeçar do zero (reset-db.js).
Um script para fazer backup das imagens do Ploc.
Um script para testar a conexão com a VPS sem precisar abrir o navegador.
Veredito: Vital. À medida que o Ploc crescer, vamos colocar nossos "robôs auxiliares" aqui.
